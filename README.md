# 📊 Retail Sales Analytics Dashboard

An interactive Business Intelligence dashboard built with **Power BI, Python, Power Query, and DAX** to analyze retail sales, profitability, product performance, and regional insights.

---

## 🚀 Tech Stack

- Power BI
- DAX
- Power Query
- Python (Pandas)
- CSV Dataset

---

## 📌 Features

- Executive KPI Dashboard
- Product Performance Analysis
- Regional Sales Insights
- Interactive Filters & Navigation
- Business Insights
